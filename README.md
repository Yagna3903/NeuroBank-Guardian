## üöÄ The Vision
Millions of Canadian seniors struggle with financial autonomy due to cognitive decline, vision loss, or motor impairments. Traditional banking apps are too complex, and phone scams are increasingly sophisticated. 

**NeuroBank-Guardian** is an embodied AI sentinel. It provides a photorealistic, empathetic interface (Azure Avatar) that seniors can talk to naturally. It doesn't just manage money; it remembers their history via **MongoDB Vector Search** and detects fraud in real-time using **Gemini 1.5 Pro**.



## üõ†Ô∏è The Tech Stack (God-Tier Integration)

* **The Brain:** `Gemini 1.5 Pro` - Used for multimodal reasoning, threat detection, and intent analysis.
* **The Face:** `Azure Real-time Synthesis Avatar` - Providing a human-like, interactive presence for low-anxiety banking.
* **The Voice:** `ElevenLabs Conversational AI` - Ultra-low latency Speech-to-Text and high-fidelity text-to-speech.
* **The Memory:** `MongoDB Atlas Vector Search` - Semantic retrieval of banking history and scam-awareness documentation.
* **The Shield:** `Solana (Devnet)` - Immutable audit logs for every AI-triggered transaction.

## üß† Key Features
1. **Adaptive Aura UI:** The interface changes color based on real-time risk assessment (Safe, Investigating, Critical).
2. **Contextual Memory:** "Did I pay my rent?" The AI uses Vector Search to verify past actions instantly.
3. **Scam Sentinel:** Detects high-pressure language or suspicious requests (like gift card payments) and blocks them.
4. **Accessibility First:** 24px+ typography, voice-first navigation, and high-contrast visual cues.
